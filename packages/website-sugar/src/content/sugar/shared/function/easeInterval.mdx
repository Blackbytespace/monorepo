# easeInterval
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">1.0.0</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -js">js</span></div>
</div>
<div class="_namespace">@lotsof.sugar.shared.function</div>
<div class="_description typo-lead typo-format">
This function allows you to call a callback function exactly as the `setInterval` would do but with an easing that you can specify,
as well as an `interval` setting to customize how many time your function will be called.
By default this function will pass the &quot;easedPercent&quot; that represent the percentage of your passed &quot;duration&quot; with the easing applied on it.
With that you can make everything you want inside your callback function.
</div>

<div class="_examples">
## Example
```js
import { __easeInterval } from '@lotsof/sugar/function';
await __easeInterval(2000, (easedPercent) => {
     // do something...
}, {
     interval: 1000 / 25 // 25 times by second
});
```
