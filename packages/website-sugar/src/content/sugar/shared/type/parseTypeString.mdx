# parseTypeString
<div class="_description typo-lead typo-format">
This method simply parse the passed typeString like &quot;string | number&quot;, or &quot;string &amp; path&quot;, etc... and return
an object defining this type string
</div>

</div>

</ol>
</div>

</ol>

## Example
```js

//   type: 'number',
//   of: undefined,
//}]
```
