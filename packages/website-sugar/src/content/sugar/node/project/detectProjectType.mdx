<div class="_description">
This function allows you to detect the project type like &quot;next&quot;, &quot;nuxt&quot;, etc...
If the project type is not detected, it will return &quot;unknown&quot; for the type and &quot;1.0.0&quot; for the version.
Here the list of detected project types:

- next
- nuxt
- astro
- remix
- sveltekit
- laravel
