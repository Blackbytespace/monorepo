# s-shade
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Declaration</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

<div class="_description typo-lead typo-format">
<p>This variable allows you to declare a color shade easily and use it in your css.
You can declare as many color shades as you want.</p>
<p>Available modifiers:</p>
<ul>
<li><code>lightness</code>: 0-100</li>
<li><code>lighten</code>: 0-100</li>
<li><code>darken</code>: 0-100</li>
<li><code>saturation</code>: 0-100</li>
<li><code>saturate</code>: 0-100</li>
<li><code>desaturate</code>: 0-100</li>
<li><code>hue</code>: 0-360</li>
<li><code>spin</code>: 0-360</li>
<li><code>alpha</code>: 0-1</li>
</ul>

</div>
<div class="_params">
## Params

<li class="_item">
<span class="_name">color<span class="_required">*</span></span><span class="_default">-</span> <span class="_type">String</span>

</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
   --s-color-main: #ff0000;

--s-shade-text: --lighten 20;
--s-shade-text-main: --lighten 35;
--s-shade-placeholder: --alpha 0.4;
--s-shade-foreground: --lighten 50;
--s-shade-background: --darken 40;
--s-shade-surface: --darken 40;
--s-shade-border: --alpha 0.2;
--s-shade-hover: --lighten 40;
--s-shade-active: --darken 10;
}

.my-element {
color: s-color(accent);
color: s-color(accent, test);
color: s-color(accent, --darken 20 --spin 30);
}

```
