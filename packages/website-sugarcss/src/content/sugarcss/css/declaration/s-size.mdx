# s-size
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Declaration</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -css">css</span></div>
</div>
<div class="_namespace">@lotsof.sugarcss.css.declaration</div>
<div class="_description typo-format typo-rhythm">
<p>This variables allows you to register a size value that you can use in your css easily.
through the s-size() function.
You can register as many size as you want.</p>

</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">value<span class="_required">*</span></span><span class="_default">-</span> <span class="_type">Number</span>
<p class="_description">The value to register for your size</p>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {

     --s-size-small: 10px;
     --s-size-medium: 20px;
     --s-size-large: 40px;
}

.my-element {
   padding: s-size(small); // 10px
   padding: s-size(medium); // 20px
   padding: s-size(large); // 40px
}
```
