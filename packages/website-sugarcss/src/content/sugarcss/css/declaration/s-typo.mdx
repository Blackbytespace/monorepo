# s-typo
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Declaration</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

<div class="_description typo-lead typo-format">
<p>This variable allows you to declare a typography easily and use it in your css.
You can declare as many typos as you want.</p>

</div>
<div class="_params">

<ol class="_list">
<li class="_item">

</li>
<li class="_item">
<span class="_name">size</span><span class="_default">&quot;1em&quot;</span> <span class="_type">Number</span>
<p class="_description">The font size to use</p>

<li class="_item">

</li>
<li class="_item">
<span class="_name">letter-spacing</span><span class="_default">0</span> <span class="_type">Number</span>
<p class="_description">The letter spacing to use</p>
</li>
<li class="_item">
<span class="_name">text-transform</span><span class="_default">&quot;none&quot;</span> <span class="_type">String</span>
<p class="_description">The text transform to use</p>

</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
  --s-typo-h1: s-font(title) s-size(h1) 1.4 0;
  --s-typo-h2: s-font(title) s-size(h2) 1.2 0 uppercase;
  --s-typo-p: s-font(default) s-size(p) 1.4 0 lowercase underline;
}

.my-element {
   @s-typo(h1);
}
```
