# s-weight
<div class="_description typo-lead typo-format">
<p>This function allows you to get a font weight from the registered ones or from the default ones that are:</p>
<ul>
<li>thin: 100</li>
<li>extra-light: 200</li>
<li>lighter: 200</li>
<li>light: 300</li>
<li>regular/normal: 400</li>
<li>medium: 500</li>
<li>semi-bold: 600</li>
<li>bold: 700</li>
<li>bolder: 800</li>
<li>extra-bold: 800</li>
<li>black: 900</li>
</ul>

</div>

## Example
```css
:root {
   --s-weight-boldeeeeeerrrrrr: 9000;
}

.my-element {
   font-weight: s-weight(bold); // 700
   font-weight: s-weight(light); // 300
   font-weight: s-weight(boldeeeeeerrrrrr); // 9000
}
```
