# s-size
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>
<div class="_platform"><span class="_platform-label">Platform:</span><span class="_platform-value -css">css</span></div>
</div>
<div class="_namespace">@lotsof.sugarcss.css.function</div>
<div class="_description typo-format typo-rhythm">
<p>This function allows you to apply a size depending on the
min, max and easing function declared using the <code>--s-sizes</code> variable, or registered with
a custom name like --s-size-small, --s-size-medium, etc...</p>

</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">...size<span class="_required">*</span></span><span class="_default">-</span> <span class="_type">Number&vert;String</span>
<p class="_description">The size&lpar;s&rpar; you want to apply</p>
</li>
<li class="_item">
<span class="_name">scalable</span><span class="_default">true</span> <span class="_type">Boolean</span>
<p class="_description">Specify if you want the size&lpar;s&rpar; to be scalable or not</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_default">-</span><span class="_type">Css</span>
<p class="_description">The generated css</p>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
     / Define min, max and a easing function
/
     --s-sizes: 0 80px;

     / Define named sizes
/
     --s-size-small: 10px;
     --s-size-medium: 20px;
     --s-size-large: 40px;
}

.my-element {
   padding: s-size(10); // 80px / 100
10 = 8px
   padding: s-size(100); // 80px / 100
100 = 80px
   padding: s-size(small); // 10px
   padding: s-size(medium); // 20px
   padding: s-size(large); // 40px
   padding: s-space(10 false); // not scalable
}
```
