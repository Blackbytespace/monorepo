# s-rem
<div class="_description typo-lead typo-format">
<p>This function allows you to convert a pixel value to a rem value.
You can pass multiple values to be converted at once.</p>

</div>

</div>

<ol class="_list">

</li>
</ol>
</div>

<ol class="_list">

## Example
```css

}
```
