<div class="_description typo-lead typo-format">
<p>This mixin allows you to create a menu icon easily.
Note that you can use the <code>s-menu-active</code> mixin to apply the active state to the menu icon.
The color applyed to the menu icon is the <code>current</code> color.</p>

## Example
```css

  &:hover {

  }
}
```
