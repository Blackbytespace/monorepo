# s-map-color
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">AtRule</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

<div class="_namespace">@lotsof.sugarcss.css.rule</div>
<div class="_description typo-lead typo-format">
<p>This rule allows you to remap a color with a new one.
This is useful when using the <code>current</code> coloe in your components.
You can then change it by using this rule like this: <code>@s-map-color(complementary, current);</code>.
This will set the <code>current</code> color to the <code>complementary</code> one.</p>

</div>

<div class="_examples">
## Example
```css
:root {
   --s-color-accent: #ff0000;
   --s-color-complementary: #00ff00;
   --s-color-current: #0000ff;
}

.my-element {
   ._title {
      color: s-color(current);
   }

   &.-error {
     @s-map-color(complementary)
   }
}
```
