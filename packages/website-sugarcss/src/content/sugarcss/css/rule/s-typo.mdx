# s-typo
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">AtRule</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

</div>
<div class="_namespace">@lotsof.sugarcss.css.rule</div>
<div class="_description typo-lead typo-format">
<p>This rule allows you to apply a registered typography easily with all his properties like:</p>
<ul>
<li><code>font</code>: The font to use from the registered ones</li>
<li><code>size</code>: The font size to use</li>
<li><code>line-height</code>: The line height to use</li>
<li><code>letter-spacing</code>: The letter spacing to use</li>
<li><code>text-transform</code>: The text transform to use</li>
<li><code>text-decoration</code>: The text decoration to use</li>
</ul>

</div>

<div class="_examples">
## Example
```css
:root {
   --s-typo-h1: s-font(sans) s-size(h1) 1.4 0;
}

.my-element {
     @s-font(h1);
}
```
