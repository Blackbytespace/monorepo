# s-zindex
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">Function</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

<div class="_description typo-lead typo-format">
This rule allows you to apply a z-index value from the registered z-indexes in the config.
</div>
<div class="_params">
## Params
<ol class="_list">
<li class="_item">
<span class="_name">name<span class="_required">*</span></span><span class="_default">-</span> <span class="_type">String</span>
<p class="_description">The z-index name you want to get from the config</p>
</li>
<li class="_item">
<span class="_name">offset</span><span class="_default">0</span> <span class="_type">Number</span>
<p class="_description">An offset to apply to the z-index value</p>
</li>
</ol>
</div>
<div class="_return">
## Return
<ol class="_list">
<li class="_item">
<span class="_default">-</span><span class="_type">Css</span>
<p class="_description">The generated css</p>
</li>
</ol>
</div>
<div class="_examples">
## Example
```css
:root {
   --s-zindex-header: 100;
   --s-zindex-chatbot: 200;
}

.my-element {
@s-zindex(header); // z-index: 200;
@s-zindex(chatbot); // z-index: 200;
@s-zindex(header, -1); // z-index: 199;
}

```
