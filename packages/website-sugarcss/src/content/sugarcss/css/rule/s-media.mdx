# s-media
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">AtRule</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>

<div class="_description typo-lead typo-format">
<p>This function allows you to define media queries easily with these features:</p>
<ul>
<li>Define your own media queries like so: --s-media-desktop: 1024px 9999px;</li>
<li>Use defined media queries easily</li>
<li>Support for <code>dark</code> and <code>light</code> media queries</li>
<li>Support for &quot;color schema&quot; media queries like <code>cs-...</code> that will target elements inside the <code>cs-...</code> class</li>
<li>Support for &quot;theme&quot; media queries like <code>theme-...</code> that will target elements inside the <code>theme-...</code> class</li>
</ul>
<p>Support for operators like:</p>
<ul>
<li><code>lt-...</code>: lower than</li>
<li><code>lte-...</code>: lower than or equal</li>
<li><code>gt-...</code>: greater than</li>
<li><code>gte-...</code>: greater than or equal</li>
<li><code>e-...</code>: equal</li>
<li><code>dark</code>: dark mode</li>
<li><code>light</code>: light mode</li>
<li>&#39;theme-...&#39;: theme</li>
<li><code>cs-...</code>: color schema</li>
</ul>

## Example
```css
:root {
   --s-media-phone: 0 767px;
   --s-media-tablet: 768px 1023px;
   --s-media-desktop: 1024px 9999px;
}

.my-element {
   @media phone { ... }
   @media lt-desktop { ... }
   @media e-tablet { ... }
   @media dark { ... }
   @media gt-phone { ... }
   @media theme-half-life { ... }
   @media theme-moon { ... }
}
```
