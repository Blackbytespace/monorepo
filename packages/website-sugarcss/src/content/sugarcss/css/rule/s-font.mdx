# s-font
<div class="_metas">
<div class="_type"><span class="_type-label">Type:</span><span class="_type-value">AtRule</span></div>
<div class="_status"><span class="_status-label">Status:</span><span class="_status-value -stable">stable</span></div>
<div class="_since"><span class="_since-label">Since:</span><span class="_since-value">0.0.1</span></div>

</div>
<div class="_namespace">@lotsof.sugarcss.css.rule</div>
<div class="_description typo-lead typo-format">
<p>This rule allows you to apply a registered font easily with all his properties like:</p>
<ul>
<li><code>family</code>: The font family like &quot;sans-serif&quot;, &quot;serif&quot;, etc...</li>
<li><code>weight</code>: The font weight like &quot;normal&quot;, &quot;bold&quot;, etc...</li>
<li><code>style</code>: The font style like &quot;normal&quot;, &quot;italic&quot;, etc...</li>
<li><code>variant</code>: The font variant like &quot;normal&quot;, &quot;small-caps&quot;, etc...</li>
<li><code>letter-spacing</code>: The font letter spacing like &quot;normal&quot;, &quot;0.1em&quot;, etc...</li>
</ul>

</div>

</li>
</ol>
</div>

</ol>
</div>
<div class="_examples">
## Example
```css
:root {
   --s-font-title: VT323 600 normal normal 0.1em;
}

.my-element {
     @s-font(title);
}
```
